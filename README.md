## Steps to start Next.js
1. `npm init`
2. `npm install next react react-dom`
3. Update the package.json
```
"scripts": {
  "dev": "next",
  "build": "next build",
  "start": "next start"
}
```
4. Start adding page to pages folder
5. `npm run dev` and checkout `https://localhost:3000`
